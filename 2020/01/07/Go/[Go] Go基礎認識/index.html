[Go] Go基礎

前言

本篇用於記錄Go一些基礎的概念，包含變數類型、迴圈及條件運算等

變數宣告

Go是一種靜態類型的語言，是強型別語言。因為型別都是固定的，就算不先宣告型別，也必須要有初始值，讓編譯器來判斷這個變數的預設值是什麼型別。

Go宣告型別是放在變數之後

短語法

Go也可以使用:=的方式，用更簡短的語法來定義變數

:=結構不能使用在函式外。

合適的使用時機

一般來說，短語法適合的使用時機有三個:

  • 已知變數的初始值
  • 讓程式更簡潔
  • 用於重複宣告

重複宣告(redeclaration)

在Go的程式設計中，是不允許同一變數宣告兩次，若重新對一變數進行宣告時會出現name redeclared in this block的錯誤訊息。

解決重複宣告的問題

可以採用宣告多個變數，也就是除了重複宣告的變數外，再另外以短語法的方式宣告一個(或多個)新變數。

初始值

Go在宣告變數時，如果沒有給定該變數初始值，Go會依照宣告變數的資料型態去給予預設的初始值，如:

  • int: 0
  • bool: false
  • float64: 0
  • string: ""

不允許已宣告但未被使用的變數

在Go的設計中，不允許已宣告但未被使用的變數，若有前面的情形，會在編譯時跳出declared and not used的錯誤訊息。
