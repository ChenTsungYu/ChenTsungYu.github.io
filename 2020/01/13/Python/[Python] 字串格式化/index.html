[Python] 字串格式化

前言

進行資料處理時，很多時候都需要對數值進行格式化轉為字串做拼接，或是某段字串與變數做串連。Python的字串格式化用於簡化靜態字串和變數的串接，並格式化變數，當然也可以對數值進行格式化成字串，字串格式化的方式有四種:

  • 百分比(%)
  • str.format => 作法:'{}'.format()
  • f-string(又作formatted string literals)
  • 樣板字串(Template String)

f-string是Python3.6之後才有的，實際上對比的話f-string是三者中(百分比(%)與'{}'.format())效能最好的，同時也提高可讀性，建議實際開發時，直接用f-string方法取代前兩者哦！

單雙引號、大括號({})與跳脫字元

f-string大括號內所用的引號不能和大括號外的引號定義的符號相衝突，可依使用情況切換''和""。
若大括號內外兩個引號相衝突，會報SyntaxError錯誤。

若""和''無法滿足需求，可以使用''' '''和""" """

backslash斜槓(\)的使用

大括號外的引號可以使用\，但大括號內不能使用\

如果在大括號內使用\，會報SyntaxError。

解決方式

先將含\的字串assign給一變數，再以變數的形式填入{}中。

上述例子即是把"I\'m Tom"這個含有特殊字元的\assign給一變數，透過解析變數的方式來避免SyntaxError。

用於多行字串

f-string採用{content:format}來設定字串格式，content是替換並填入字符串的內容，可以是變數、表達式(運算子亦可)或函數，format是格式設定。預設格式為{:format}。

讓跳脫字元失效

若要讓跳脫字元失效，在f指令前再加上r，表示raw(原始字元)
