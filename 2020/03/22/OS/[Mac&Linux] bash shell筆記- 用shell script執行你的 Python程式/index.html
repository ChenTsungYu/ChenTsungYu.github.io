接著執行./echo.py

Hashbang

是一個由井號(#)和驚嘆號(!)構成的字元，需寫在文字檔案的第一行的前兩個字元，作業系統的程式載入器會分析Hashbang後的內容，將這些內容作為直譯器指令，並呼叫該指令。 如: 以指令#!/bin/bash開頭的檔案在執行時會實際呼叫位於/bin/bash程式。
