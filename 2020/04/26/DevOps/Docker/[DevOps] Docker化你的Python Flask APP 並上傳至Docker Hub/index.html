語法回顧

  • FROM [Docker Image Name:TAG]
    指定這個映像檔要以哪一個Image為基底來建構
  • WORKDIR 設定工作目錄
  • COPY 複製本地端的檔案/目錄到映像檔的指定位置中
  • RUN Build Image 時會執行的指令，下載相依套件
  • EXPOSE container 啟動時會監聽的port
  • CMD 執行 Container 時的指令
