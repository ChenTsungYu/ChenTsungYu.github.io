еүҚиЁҖ

е»әз«ӢSecurity Group

еұ¬жҖ§еҸҜи©іиҰӢе®ҳж–№зөҰзҡ„sample

дҫқз…§дёҠиҝ°зЁӢејҸзўјеҹ·иЎҢterraform applyпјҢеҚіеҸҜжҲҗеҠҹе»әз«ӢSecurity Group

жңүй—ңSecurity GroupиЁӯе®ҡзҡ„ruleпјҢзҜ„дҫӢзҡ„иЁӯе®ҡдёҚжҳҜжҜ”ијғе®үе…Ёзҡ„еҒҡжі•пјҢеҰӮingressиЁӯе®ҡгҖӮ
NOTE: Setting protocol = вҖңallвҖқ or protocol = -1 with from_port and to_port will result in the EC2 API creating a security group rule with all ports open. This API behavior cannot be controlled by Terraform and may generate warnings in the future.
дёҠиҝ°зӮәе…¶дёӯдёҖеҖӢжңүй—ңе®үе…ЁиЁӯе®ҡзҡ„жҸҗйҶ’пјҢжӣҙеӨҡе®үе…Ёе»әиӯ°еҸҜеҸғиҖғе®ҳж–№ж–Үд»¶

еӢҷеҝ…иҰҒзңӢжё…жҘҡз•¶еүҚVPCжүҖеңЁзҡ„RegionпјҢеҲҘеғҸжҲ‘дёҖжЁЈз¬¬дёҖж¬ЎеҜҰдҪңжҠҠдёҚеҗҢRegionзҡ„VPC IDиІјдёҠеҺ»пјҢзөҗжһңдёҖзӣҙи·іеҮәThe vpc ID does not existзҡ„йҢҜиӘӨпјҒ

еҲ°AWS EC2зҡ„з®ЎзҗҶд»ӢйқўпјҢе·Ұж–№еҲ—иЎЁжүҫеҲ°Security GroupsеҚіеҸҜзңӢиҰӢеүӣеүӣж–°е»әзҡ„Security Group

дёӢијүkey pair

иҮіAWS EC2зҡ„з®ЎзҗҶд»ӢйқўпјҢе·Ұж–№еҲ—иЎЁжүҫеҲ°Key PairsпјҢйҖІе…Ҙй Ғйқўй»һж“ҠCreate key pairгҖӮ
еҸ–еҗҚзӮәec2-terraformпјҢйҒёж“ҮpemжӘ”

е®ҢжҲҗе»әз«Ӣзҡ„еҗҢжҷӮпјҢжңғдёӢијүдёҖзө„key pairеҲ°жң¬ең°пјҢе°ҮжӯӨжӘ”з§»е…Ҙе°ҲжЎҲиіҮж–ҷеӨҫдёӯпјҢжҺҘи‘—жӣҙж”№жӘ”жЎҲж¬Ҡйҷҗ

chmod 400иЎЁзӨәеҸӘжңүж“ҒжңүиҖ…жүҚиғҪеӨ дҝ®ж”№зҡ„ж¬ҠйҷҗпјҢйқһж“ҒжңүиҖ…еҸӘиғҪеӨ и®ҖеҸ–

е»әз«ӢEC2

жҺҘзәҢдёҠйқўзҡ„зҜ„дҫӢзЁӢејҸзўјпјҢжҺҘи‘—ж–°еўһaws_instanceйҖҷеҖӢresource

й»һж“ҠLaunch InstanceпјҢйҒёж“ҮAMIзӮәAmazon Linux 2 AMIпјҢInstance TypeзӮәt2.micro
AMIе°ҚжҮүidзҡ„йғЁеҲҶеҸҜд»ҘеңЁChoose AMIзҡ„з®ЎзҗҶд»ӢйқўдёӯзңӢеҲ°жҜҸеҖӢIAMе°Ҳеұ¬зҡ„ID

иҖҢInstance Typeзҡ„йғЁеҲҶеҸҜд»ҘеҸғиҖғAWSжҸҗдҫӣзҡ„typeпјҢдёҚдёҖе®ҡиҰҒжҢүз…§дёҠиҝ°зҜ„дҫӢ

vpc_security_group_idsзҡ„и©ұпјҢе…¶еҜҰеҸҜд»Ҙеҫһ .tfstate жӘ”жЎҲдёӯжүҫеҲ°security group зҡ„idпјҢдёҚйҒҺе»әиӯ°йқҲжҙ»дёҖй»һпјҢдёҚиҰҒзӣҙжҺҘеҜ«жӯ»пјҢйӮ„жҳҜйҖҸйҒҺaws_security_group.http_server_sg.idдҫҶеҸ–еҫ—idжңғжҜ”ијғеҘҪгҖӮ

subnet_idзҡ„и©ұпјҢеӣһеҲ°VPCз®ЎзҗҶд»ӢйқўпјҢе·Ұж–№ж¬„дҪҚжңүsubnetпјҢеҸҜжүҫеҲ°жғіиҰҒзҡ„subnet id

зўәиӘҚжІ’е•ҸйЎҢд№ӢеҫҢеҸҜд»Ҙе…ҲдёӢterraform validateйҖІиЎҢй©—иӯүпјҢжҺҘи‘—еҹ·иЎҢterraform applyе°ұеҸҜд»ҘеӣүпјҒ

йҖЈжҺҘEC2

зҸҫеңЁиҰҒйҖІиЎҢйҒ з«ҜйҖЈжҺҘEC2пјҢжҲ‘зҡ„дҪңжі•еңЁе°ҲжЎҲзӣ®йҢ„дёӢж–°еўһдёҖиіҮж–ҷеӨҫпјҢеҗҚзӮәaws_key_pairпјҢдёҰеҺҹе…ҲдёӢијүзҡ„key pairз§»е…ҘжӯӨиіҮж–ҷеӨҫдёӢпјҢеҶҚиЁӯдёҖи®Ҡж•ёaws_key_pairпјҢиЈЎйқўеӯҳж”ҫkey pairзҡ„дҪҚзҪ®и·Ҝеҫ‘гҖӮ
жҺҘи‘—ж–°еўһresourceиЁӯе®ҡconnectionеұ¬жҖ§гҖӮ

  • type = "ssh": иЎЁжҲ‘еҖ‘жҺЎз”ЁSSHзҡ„ж–№ејҸйҖЈжҺҘ
  • host = self.public_ip: иЎЁдё»ж©ҹе°ҚжҮүзҡ„ip
  • private_key = file(var.aws_key_pair): з§Ғй‘°еҫһжң¬ең°е°ҲжЎҲеӨҫе…§йҖІиЎҢи·Ҝеҫ‘жҗңе°ӢпјҢжүҫеҲ°дёӢијүдёӢдҫҶзҡ„.pemжӘ”

жҺҘи‘—иЁӯе®ҡprovisionerеұ¬жҖ§пјҢ"remote-exec"иЎЁйҖІиЎҢйҒ з«Ҝж“ҚдҪңдёӢйҒ”зҡ„жҢҮд»ӨпјҢзөҰе®ҡinlineйҷЈеҲ—пјҢиЈЎйқўеӯҳж”ҫиҰҒдёӢйҒ”зҡ„жҢҮд»ӨпјҢжҜҸиЎҢжҢҮд»Ө,еҚҖйҡ”

ж–°еўһйҖЈжҺҘEC2зҡ„зЁӢејҸзўјеҫҢпјҢй Ҳе…Ҳе°ҮEC2 terminateжүҚжңғеҹ·иЎҢremote-execзҡ„жҢҮд»ӨпјҢжүҖд»Ҙе…Ҳеҹ·иЎҢterraform destroyпјҢеҶҚеҹ·иЎҢterraform apply

еңЁзөӮз«Ҝж©ҹдёҠеӨ§жҰӮжңғзңӢеҲ°дёҠиҝ°зҡ„еҹ·иЎҢз•«йқўгҖӮ
жҺҘи‘—еҫһ.tfstateжӘ”дёӯжүҫеҲ°public_dnsе°ҚжҮүзҡ„з¶ІеқҖпјҢиІјдёҠз¶ІеқҖд№ӢеҫҢе°ұеҸҜд»Ҙй ҶеҲ©иЁӘе•Ҹеүӣеүӣзҡ„е…§е®№еӣүпјҒ

йҖІйҡҺпјҡйҒёз”Ёй җиЁӯзҡ„VPC

зӣҙжҺҘдҪҝз”Ёй җиЁӯ(default)зҡ„VPCпјҢе°Үе®ғж”№зӮәеӢ•ж…Ӣзҡ„пјҢз”ЁдҫҶеҸ–д»ЈеҺҹжң¬еҜ«жӯ»зҡ„vpc_idпјҢTerraformжңғиҮӘеӢ•ж №ж“ҡз•¶еүҚProviderиЁӯе®ҡзҡ„RegionпјҢжүҫи©ІRegionдёӢй җиЁӯзҡ„VPC IDпјҢйҖҷжЁЈе°ұдёҚжңғзҷјз”ҹеғҸжҲ‘дёҠиҝ°зҠҜдёӢзҡ„йҢҜиӘӨ: жүҫйҢҜRegionзҡ„VPC ID
еҹ·иЎҢдёӢж–№жҢҮд»Ө

дёҰдҝ®ж”№vpc_idзөҰе®ҡзҡ„еҖј

еҹ·иЎҢдёӢж–№жҢҮд»Ө

е®Ңж•ҙзЁӢејҸзўј

еҸҰеӨ–еҖјеҫ—дёҖжҸҗзҡ„жҳҜпјҢе®ҳж–№жңүжҸҗеҲ°aws_default_vpcдёҚеҗҢж–је…¶д»–зҡ„resourceпјҢTerraformдёҰдёҚжңғйЎҚеӨ–е»әз«ӢйҖҷй …иіҮжәҗпјҢиҖҢжҳҜзӣҙжҺҘе°Үе®ғзҙҚе…Ҙз®ЎзҗҶгҖӮ
