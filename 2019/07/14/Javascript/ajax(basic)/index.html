[JavaScript] Ajax

зӯҶиЁҳAjaxеүҚпјҢиҰҒе…ҲзҗҶи§Јз¶Із«ҷзҡ„дё»еҫһжһ¶ж§Ӣпјҡе®ўжҲ¶з«Ҝе’ҢжңҚеӢҷеҷЁз«Ҝ

е®ўжҲ¶з«Ҝ(Client-side)пјҡ жҢҮиЁӘе®ўзҡ„йӣ»и…Ұе’ҢзҖҸиҰҪеҷЁ

дјәжңҚеҷЁз«Ҝ(Server-side): еӣһжҮүе®ўжҲ¶з«Ҝи«ӢжұӮзӮәдјәжңҚеҷЁз«Ҝ

иҲүеҖӢдҫӢеӯҗпјҢд»ҠеӨ©з•¶дҪҝз”ЁиҖ…з”ЁзҖҸиҰҪеҷЁйҖЈз·ҡдёҠдҪ зҡ„ ISP (з¶Іи·ҜдҫӣжҮүе•Ҷ e.g. дёӯиҸҜйӣ»дҝЎ)йҖ иЁӘз¶Із«ҷ(е°Қз¶Із«ҷжҸҗеҮәи«ӢжұӮ)пјҢе°Үз¶Із«ҷдјәжңҚеҷЁдёҠзҡ„иіҮж–ҷеҸҠзЁӢејҸзўјдёӢијүеҲ°жң¬ең°з«Ҝ(дҪҝз”ЁиҖ…йӣ»и…Ұ)пјҢдёҰеңЁзҖҸиҰҪеҷЁдёҠеҒҡе‘ҲзҸҫгҖӮ

дёӢйқўз•«ејөең–иЎЁеҒҡиӘӘжҳҺ:

еҘҪпјҢзҸҫеңЁзҹҘйҒ“йҖ иЁӘз¶Ій Ғзҡ„йҒҺзЁӢдёӯжңғзҷјз”ҹд»ҖйәјдәӢжғ…д№ӢеҫҢпјҢиҰҒдәҶи§ЈиҰҒеҰӮдҪ•и·ҹServerжӢҝиіҮж–ҷпјҢзҷјйҖҒRequestгҖӮ

з”ЁJavascriptеҜ«з¶Ій Ғзҡ„жҷӮеҖҷпјҢеҸҜд»ҘзҷјзҸҫJavascriptжҳҜдёҖиЎҢдёҖиЎҢеҹ·иЎҢпјҢйҖҷжЁЈзҡ„зү№жҖ§зЁұдҪңвҖқеҗҢжӯҘвҖқгҖӮ

еҗҢжӯҘзҡ„ж„ҸжҖқжҳҜJavascriptеҹ·иЎҢеҲ°жҹҗдёҖиЎҢзҡ„жҷӮеҖҷпјҢжңғзӯүйҖҷиЎҢеҹ·иЎҢе®Ңз•ўпјҢжүҚеҹ·иЎҢеҲ°дёӢдёҖиЎҢпјҢзўәдҝқеҹ·иЎҢй ҶеәҸгҖӮ

жҸӣеҸҘи©ұиӘӘпјҢз•¶д»ҠеӨ©JavascriptзҷјеҮәдёҖеҖӢRequestжҷӮпјҢжңғзӯүеҫ…ResponseеӣһдҫҶпјҢеңЁеӣһдҫҶд№ӢеүҚпјҢJavascriptеј•ж“ҺжҳҜдёҚжңғеҒҡд»»дҪ•еӢ•дҪңзҡ„пјҒ

е•ҸйЎҢдҫҶдәҶпјҢзүҪж¶үеҲ°з¶Іи·Ҝж“ҚдҪңжҷӮпјҢз•¶з¶Іи·ҜдёҠжңүеӨҡеҖӢи«ӢжұӮпјҢеҰӮжһңйӮ„жҳҜз¶ӯжҢҒеҗҢжӯҘзҡ„ж–№ејҸеҹ·иЎҢзЁӢејҸзўјпјҢиҰҒжҺҘж”¶еҲ°дёҠдёҖеҖӢResponseжүҚиғҪйҖҒеҮәдёӢдёҖеҖӢRequest

еҸҜд»Ҙй җжңҹеҲ°и®Ҡеҫ—йқһеёёиҖ—жҷӮй–“пјҢдёҚз©©е®ҡзҡ„ж“ҚдҪңе°ұйңҖиҰҒз”ЁеҲ°вҖқйқһеҗҢжӯҘвҖқзҡ„ж–№ејҸгҖӮ

йқһеҗҢжӯҘжҳҜд»Җйәјж„ҸжҖқе‘ўпјҹ

е°ұжҳҜеҹ·иЎҢе®Ңд№ӢеҫҢе°ұдёҚз®Ўе®ғдәҶпјҢдёҚзӯүзөҗжһңеӣһдҫҶе°ұз№јзәҢеҹ·иЎҢдёӢдёҖиЎҢгҖӮ

иҲүеҖӢдҫӢеӯҗпјҡ

д»ҠеӨ©иҰҒеҺ»дёҖй–“е°Ҹеҗғеә—еҗғйЈҜпјҢй»һе®ҢйӨҗеҫҢеңЁиҸңе–®дёҠеҜ«дёӢиҮӘе·ұжүҖеңЁзҡ„жЎҢиҷҹпјҢе°ҮиҸңе–®дәӨзөҰж«ғеҸ°(зҷјйҖҒи«ӢжұӮ)и®“иҖҒй—ҶзҹҘйҒ“дҪ еқҗе“ӘдёҖжЎҢпјҢй»һйӨҗе®Ңз•ўдёҚйңҖиҰҒз«ҷеңЁеә—й–ҖеҸЈзӯүйӨҗй»һеҒҡеҘҪ(дёҚз”ЁзӯүResponse)пјҢеҸҜд»ҘеӣһеҲ°иҮӘе·ұдҪҚзҪ®еҒҡиҮӘе·ұзҡ„дәӢ(зҷјеҮәе…¶д»–Request)пјҢиҖҒй—ҶиҮӘеӢ•жҠҠеҒҡеҘҪзҡ„йӨҗй»һйҖҒйҒҺдҫҶ(ResponseеӣһдҫҶ)гҖӮ

жңҖеҫҢжҲ‘еҖ‘е°ұйңҖиҰҒйҖҸйҒҺajaxж“ҚдҪңдҫҶйҒ”еҲ°йқһеҗҢжӯҘзҡ„зӣ®зҡ„пјҒ

д»ҖйәјжҳҜ Ajaxпјҹ

е…ЁеҗҚжҳҜгҖҢAsynchronous JavaScript and XMLгҖҚ, зҝ»иӯҜпјҡJavascriptе°ҚXML(иЁ»)зҡ„з•°жӯҘи«ӢжұӮж“ҚдҪңгҖӮ

йҮҚй»һеңЁж–јпјӮAsynchronousпјӮйҖҷеҖӢеӯ—пјҢеҜҰзҸҫпјӮйқһеҗҢжӯҘпјӮзҡ„ж“ҚдҪңзҚІеҸ–ж•ёж“ҡе…§е®№гҖӮ

еҸҰеӨ–пјҢserverз«ҜеӮіијё90%жңғжҳҜJSONж јејҸзҡ„жӘ”жЎҲпјҢдёӢйқўзЁӢејҸзўјжңғз”ЁJSONжӘ”еҒҡдҫӢеӯҗпјҲй»һжӯӨйҖЈзөҗеҸғиҖғи§ЈйҮӢJSONпјү

жҲ‘еҖ‘еҸҜд»ҘйҖҸйҒҺ йҖҸйҒҺXMLHttpRequestйҒ з«Ҝж’ҲеҲҘдәәзҡ„иіҮж–ҷгҖӮ

йҒҺзЁӢ:

  1. жү“й–ӢзҖҸиҰҪеҷЁ -> еүөе»әе°ҚиұЎ: XMLHttpRequest()

  2. ијёе…Ҙз¶ІеқҖ -> open()ж–№жі•

open()йңҖеё¶е…Ҙ3еҖӢеҸғж•ё:

a. ж–№жі•(method): get/post

b. url

c. async(з•°жӯҘ)

3.enter(зҷјйҖҒrequest) -> send()

4.жҺҘж”¶ж•ёж“ҡ request.onreadystatechange = functionпјҲпјү

еҸғж•ёпјҡжҳҜеӮійҒһзөҰжңҚеӢҷеҷЁзҡ„е…·й«”ж•ёж“ҡпјҢдҫӢеҰӮзҷ»е…Ҙзҡ„еёіиҷҹеҜҶзўјпјӣ

еёёз”Ёи«ӢжұӮж–№ејҸ: get/post

getи«ӢжұӮпјҡ

еңЁи«ӢжұӮURL(з¶ІеқҖ)еҫҢйқўд»ҘпјӮ?пјӮзҡ„еҪўејҸи·ҹдёҠзҷјзөҰжңҚеӢҷеҷЁзҡ„еҸғж•ёпјҢеӨҡеҖӢеҸғж•ёд№Ӣй–“з”Ё&йҡ”й–Ӣ

? еҫҢйқўи·ҹзҡ„жҳҜиҰҒеӮізөҰserverзҡ„еҸғж•ё

& еҸғж•ёд№Ӣй–“зҡ„й–“йҡ”

post и«ӢжұӮпјҡ

зҷјзөҰжңҚеӢҷеҷЁзҡ„еҸғж•ёе…ЁйғЁж”ҫеңЁи«ӢжұӮй«”дёӯпјҲURL дёӯзңӢдёҚеҲ°пјү

гҖҗжіЁпјҡpost еӮійҒһзҡ„ж•ёж“ҡйҮҸжІ’жңүйЎҜзӨәпјҢе…·й«”еҫ—зңӢжңҚеӢҷеҷЁзҡ„иҷ•зҗҶиғҪеҠӣгҖ‘

Httpи«ӢжұӮе ұж–ҮеҲҶ3йғЁеҲҶ

1.и«ӢжұӮиЎҢ 2.и«ӢжұӮй ӯ 3.и«ӢжұӮй«”

*readystateеұ¬жҖ§:

Ajaxи«ӢжұӮиҝ”еӣһзҡ„ж•ёж“ҡеҚіеӯҳж”ҫеңЁи©Іеұ¬жҖ§д№ӢдёӢ(иҝ”еӣһеӯ—дёІ)

0: еҲқе§ӢеҢ–,йӮ„жІ’иӘҝз”Ёopen()ж–№жі•

1: ијүе…Ҙ,е·ІиӘҝз”Ёsend()ж–№жі•,жӯЈеңЁзҷјйҖҒrequest

2: ијүе…Ҙе®ҢжҲҗ,send()ж–№жі•е®ҢжҲҗ,е·Іж”¶еҲ°е…ЁйғЁresponse(йҹҝжҮүе…§е®№)

  1. и§Јжһҗ,жӯЈеңЁи§Јжһҗresponse(еҫһserverз«Ҝиҝ”еӣһзҡ„ж•ёж“ҡ)

4.е®ҢжҲҗ,responseйҹҝжҮүе…§е®№и§Јжһҗе®ҢжҲҗ,еҸҜеңЁclientз«ҜиӘҝз”Ё

зҸҫеңЁйҖҷеҖӢз¶Із«ҷзҡ„з¶Із«ҷгҖҢдёҚеҗҢжәҗгҖҚзҡ„жҷӮеҖҷпјҢзҖҸиҰҪеҷЁдёҖжЁЈжңғзҷј RequestпјҢдҪҶжҳҜжңғжҠҠ Response зөҰж“ӢдёӢдҫҶпјҢJavaScript жӢҝеҲ°дёҰдё”еӮіеӣһйҢҜиӘӨгҖӮ

дёҚеҗҢжәҗжҳҜд»Җйәјж„ҸжҖқпјҹпјҹпјҹз°ЎиЁҖд№ӢпјҢеҸӘиҰҒжҳҜз¶ІеҹҹпјҲ Domainпјү дёҚдёҖжЁЈе°ұжҳҜдёҚеҗҢжәҗпјҢз¶ІеқҖй–Ӣй ӯhttpе’Ңhttpsд№ҹдёҚеҗҢжәҗпјҢз«ҜеҸЈиҷҹ(port)дёҚдёҖжЁЈд№ҹдёҚеҗҢжәҗгҖӮзӣёй—ңзҡ„и§Јжұәж–№жЎҲ
