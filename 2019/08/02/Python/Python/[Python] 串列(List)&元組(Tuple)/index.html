[Python] 串列(List)&元組(Tuple)

前言

串列是夠提供儲存資料的記憶體空間，Python中的串列(List)類似其他程式語言中的陣列(Array)，是一個有順序性的元素集合，若要可根據list中的實際內容或是該元素在list中的位置用索引值(index)進行查找。

串列(List)

串列內元素的資料型態可以相同或不同，甚至可以是串列

  • 注意索引值是從0開始計

索引值不可超出串列範圍，否則會產生錯誤

負索引值也同樣不可超出串列範圍

取得串列長度

使用len()方法，回傳該串列的長度，得知串列內包含多少元素。

其中:

  • start: 開始索引的位置，預設的索引位置為第一個
  • end: 結束索引的位置，預設的索引位置為最後一個
  • sep: 索引的間隔，預設為1

範圍取值更多的應用

若範圍取值的start被省略，Python會自動選取預設的索引位置 (第一個;index=0) ，而範圍取值的end被省略，則自動選取預設的索引位置 **(最後一個;index=-1)**。

取出來的值只會包含開頭，不會包含結束的index

取得最後一個元素

若想要取得list中的最後一個元素，index值輸入-1。以此類推，若要選擇倒數第二個位置的元素，index值則填入-2。

複製串列

省略start及end兩個索引值，只留一個冒號在中括號中，如[:]就可以複製整個串列。

重複n次

list1 * n方法，符號使用*，次數為n。

多維串列內元素的存取

使用多個中括號[]組合來完成。如:my_list[index][index]

增加元素

append()

使用append() 方法將新元素增加到list最後面。

insert()

指定所增加元素的位置，使用insert() 方法，結構會是:myList.insert(index, element)。

  • index: 表插入位置
  • element: 表放入的元素

extend()

extend() 方法可以一次加入很多個值或將某個list中的元素加到另一個list。

注意：如果是要一次添加多個元素，不能使用insert()或append()方法。

簡單來說，前兩者是以整個list 的資料型態被塞入，而不是以元素的形式被加入。

List合併

Python在合併串列上提供一個簡單的方法: +，與字串拼接用法相同，將多個串列用+進行合併。

移除元素

remove()

如果不確定或不在意元素在list中的位置，可以使用remove()根據指定的值來刪除元素。

pop()

藉由索引值位置來指定刪除的元素，且會回傳被刪除的值。結構為myList.pop(index)

若pop內省略index值的話，預設為刪除最後一個元素，

del

刪除指定位置元素，使用del方法可以刪除一個元素，當元素刪除之後，位於它後面的元素會自動往前填補空出來的位置。

排序

使用sort()方法對list內的元素進行排序，預設情況下，字串會照字母順序排列、數字則會是遞增排列。

字母順序排列

數字遞增排列

sort()方法會改變原始list

不改變原始list的方法複製一個排序過的list

若想要保存完整的原始資料，但同時又需要有被排序過的資料。可以使用sorted方法。

排序反轉

數字遞減排列、或是依照字母順序的反向排列。

括號中()可以設定排序的條件

在原本的sort()內加上參數reverse=True即可。

反轉

把 list內的元素順序反轉可以使用reverse()，會照當前的順序進行反轉。

查找/檢測/計算

找極大(max())、min(sum())與計算list內元素的總和(sum())

查找某個元素在list內的索引值

採用.index(element)的方法，element為要查找的元素，回傳值為該元素的在list中的位置(index值)

如果輸入的元素並不包含在list裡面的話，會出現ValueError

檢測某值是否不在串列中：in、not in

list可搭配Python中的運算元in判斷目標元素是否在該list裡面，回傳值為True或False的bool值。
若運算元為not in，判斷後的回傳結果與in相反。

同時取得目標元素值與index值

使用enumerate()方法，配合for loop進行實作。

index值可以省略，回傳值為tuple的資料結構

更改index起始值

enumerate() 內加上start參數並給index值即可。

list轉換為字串

使用.join()方法，格式為newList = "seperator".join(listName)，其中seperator替換成要隔開的符號。

把字串分割成list

在原有的字串上使用.split()做分割，預設以空白做分割的符號。

元組(Tuple)

Tuple就是**不可做修改的List**，元組的結構與串列完全相同，差別在於元素個數及元素值皆不能改變，但串列可以。語法結構會是:元組名 = (元素1, 元素2,.... )

範例

若發生tuple內的元素被修改會出現TypeError的錯誤資訊。

Tuple好處:

  • 執行速度比List快: 因為內容不會改變，因此Tuple內部結構比List簡單，執行速度較快。
  • 存於Tuple的資料較為安全: 因為內容不會改變，不會因程式設計疏忽而變更資料內容。
