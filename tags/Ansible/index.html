[Ansile] 入門概念

什麼是 Ansile?

Ansile 是由 Red Hat(紅帽) 公司所提供，經開放原始碼社群進行開發，秉持基礎設施即代碼為其中一項理念，專注在 IT 自動化領域，透過撰寫 YAML 腳本對 IT 基礎設施進行操作及管理。

主要的優點：

  • 以 YML 格式編寫，容易上手與維護
  • 無代理(Agentless)程式
  • 透過 SSH 進行連線，用於與遠端 Server 溝通。
  • 不需要安裝在 Client 端
