[JavaScript] Array

前言

JavaScript 的陣列可以看作是一種特別的「物件」，透過 typeof會返回陣列的類型是object。陣列是一組按次序排列的值， 放在裡面的東西稱為元素，每個元素都有其位置，稱為索引，找到索引值，就可以得知該位置元素的值。

  • 陣列只能透過 [] 來存取
  • 索引值從0開始排序，最後一個索引值為 array.length-1

陣列內可以是原始的資料類型、其他陣列、函式等等。
